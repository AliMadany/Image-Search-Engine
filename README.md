# Image-Search-Engine

 The rapid growth of digital media every day is the driving force behind this thesis. My aim
 is to build an efficient and accurate reverse image search engine that helps users find what
 they are looking for using a picture and input search. Significant progress has already been
 made in this field to create reliable and precise models for search engines. However, with
 new media being shared daily, it is challenging for current algorithms to keep up. New
 models need to be developed to match the rise in media sharing. This thesis investigates
 the problem in more detail and proposes innovative solutions. With computer vision as
 the core methodology, it focuses on advanced algorithms and methods for image retrieval
 and feature extraction. The results of this thesis pave the way for more complex work
 in the future and advancements in machine learning for better models. The knowledge
 gathered from this research can be applied in various fields, such as content-based image
 retrieval, surveillance, security, the medical field, and more. Specifically, the use of deep
 learning techniques and neural networks is explored to enhance the accuracy and speed of
 reverse image searches. Additionally, the thesis examines the integration of metadata and
 contextual information to improve search relevance, offering a comprehensive approach
 to tackling the challenges faced by current reverse image search technologies
